Data Taxonomy

The taxonomy harmonises multimodal neuroscience and clinical datasets using interoperable identifiers to support analytics and governance.

Primary Entities
Entity Description Standards
Participant Profile Demographics, consent status, longitudinal identifiers. HL7 FHIR · ISO/TS 17975
Neuroimaging Session Acquisition metadata, modality, preprocessing lineage. BIDS · DICOM
Clinical Assessment Clinician-reported outcomes, cognitive assessments. LOINC · SNOMED CT

Ontologies and Standards

SNOMED CT neurological disorders subset aligned with internal taxonomy; validation with clinical advisors scheduled for 2026 Q1.

Ontology revisions follow semantic versioning. Deprecations require migration notes and crosswalk tables published alongside release tags.
