Knowledge Architecture Reference

This documentation outlines the taxonomies, ontologies, and documentation framework guiding data stewardship for the platform.

Data Taxonomy

The taxonomy harmonises neuroimaging, electrophysiology, and clinical assessment data into a unified schema to support integrative analytics.

Core Data Entities
Entity Description Primary Standards
Participant Profile Demographics, consent status, and longitudinal participation identifiers. HL7 FHIR, ISO/TS 17975
Neuroimaging Session Acquisition parameters, modality metadata, preprocessing status. BIDS, DICOM
Clinical Assessment Structured clinician-reported outcomes and validated instrument scores. LOINC, SNOMED CT

Ontologies and Standards

Ontology alignment ensures interoperability across domains. Controlled vocabularies are curated through quarterly governance review.

Status: Draft alignment complete for SNOMED CT neuro disorders subset; pending validation with clinical advisors.

Versioning Strategy

Each ontology ingestion is versioned semantically (major.minor.patch) and mapped to release tags in the repository. Deprecations include migration guidance.
